# Demo Script — FGMC2 KMIS Prototype

## Flow A: "Single Place for Documents"

1. Start as **Internal Admin** (select from landing page or role switcher).
2. Navigate to **Upload & Validate**.
3. Click **"add mock files for demo"** to add 3 sample files.
4. Click **"Next: Assign Metadata"** — select countries, themes, and other metadata.
5. Click **"Submit as Draft"** — files appear in the Validation Queue.
6. Click **"Validate"** on one or more draft documents.
7. Switch role to **Internal Viewer**.
8. Navigate to **Documents** — the validated documents are now visible.
9. Use **search bar** and **filters** to find specific documents.

## Flow B: "Ask a Question in Plain Language"

1. As **Internal Viewer** or **Admin**, navigate to **AI Q&A**.
2. Optionally set scope filters (country, theme) in the left panel.
3. Click a **suggested prompt chip** or type your own question.
4. View the AI response: summary paragraph, bullet points, and source references.
5. Click a **source document link** to navigate to the document detail page.
6. Toggle **"Show reference snippets"** to expand/collapse source quotes.

## Flow C: "Living Country/Thematic Pages"

1. Navigate to **Evidence Pages**.
2. Switch between **Countries** and **Themes** tabs.
3. Click on **Ghana** (or any country/theme card).
4. View the **timeline of updates** with dates, tags, and source links.
5. Review the **Risk Snapshot** panel on the right.
6. As **Admin**, click **"+ Add Update"** to create a new timeline entry.
7. Fill in title, body, tags, and link source documents, then save.

## Flow D: "CoP Sharing (Read-Only)"

1. As **Internal Admin**, navigate to **CoP Publishing**.
2. Find a validated document in the "Ready to Publish" list.
3. Click **"Publish to CoP"** — choose section and add description, then confirm.
4. Switch role to **External CoP Member**.
5. Browse the **CoP Home** page with navigation cards.
6. Navigate to **Learning Library** — the published document is visible.
7. Use filters to search by theme, country, or document type.
